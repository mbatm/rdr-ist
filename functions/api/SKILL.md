# kayserim.net Haber Yazım Becerisi

## Genel Tanım
Bu beceri, kayserim.net için 1ha.com.tr kaynaklı haberlerin Türkiye yerel gazetecilik diline uygun şekilde yazılmasını sağlar. Analiz edilen 100+ kayserim.net haberi ve 1ha haber akışı esas alınmıştır.

---

## TEMEL İLKELER

### 1. Haber Dili Kuralları
- **Tarafsız, raporlama odaklı** yazım — editöryal yorum YASAK
- **5N1K** (Ne, Nerede, Ne zaman, Nasıl, Neden, Kim) ilk paragrafta tamamlanır
- **Spekülasyon yasak** — "olmuş olabilir", "muhtemelen" ifadeleri kullanılmaz
- **Genel tavsiye cümleleri yasak** — "benzer olayları önlemek için...", "dikkat edilmesi gerekmektedir..."
- **Dolgu metin yasak** — yapay uzatma, anlamsız tekrar kabul edilmez
- **Edilgen yapı tercih edilir** — "yakalandı", "gözaltına alındı", "sorgulandı"

### 2. Kayserim.net'e Özgü Kimlik
- "Kayseri'de", "Kayseri merkezli", "İlimizde" ifadeleri kullanılır
- Kayseri ilçe adları doğru yazılır: Melikgazi, Kocasinan, Talas, Hacılar, İncesu, Develi, Pınarbaşı vb.
- Önemli lokasyonlar: OSB, Mimarsinan OSB, Erciyes, Sivas Bulvarı, Ankara Caddesi, Cumhuriyet Meydanı

## BAŞLIK (TITLE) FORMATLARI

### Asayiş / Trafik Haberleri
```
[Kişi/Araç sayısı + olay]: [Sonuç]
→ "Otomobil kamyonun altına girdi: 3 yaralı"
→ "35 araç ve 86 şahıs sorgulandı"
→ "Kayseri'de 399 aranan şahıs yakalandı"
→ "22 yıl hapis cezası bulunan firari yakalandı"
```

### Açıklama / Konuşma Haberleri
```
[Ad Soyad]; "[Alıntı başlangıcı...]"
→ "Binali Yıldırım; 'Kayseri'yi her zaman bir adım daha ileri gitmiş olarak buluyoruz'"
→ "Vali Yardımcısı Kaçmaz, 'İş bulma gayretinde olan vatandaşlarımıza sağlıklı ortam sağlamayı amaçlıyoruz'"
NOT: Alıntı başlıkta 60 karakteri geçiyorsa "..." ile kısaltılır
```

### Operasyon / Büyük Haber
```
[Eylemli güçlü ifade], [sayısal sonuç]
→ "Kayseri'den düğmeye basıldı, 8,6 milyarlık bahis ağı çöktü"
→ "Kayseri dahil 64 ilde siber suç operasyonu: Bin 21 şüpheli yakalandı"
```

### Güncel / Genel
```
[Konu] masada: [Detay]
→ "TBMM'de çocuk suçları masada: Ceza yaşının 10'a düşürülmesi önerildi"
→ "Gram altın 6 bin 527 TL ile son iki ayın en düşük rakamını buldu"
```

## SAYI VE RAKAM YAZIMI
Türk haber geleneğine uygun:
- "119 şüpheli" → değil "yüz on dokuz"
- "8,6 milyar" → büyük rakamlar kısaltılır
- "bin 21 şüpheli" → bin + rakam formatı (AA/DHA stili)
- "28'nci TBMM Başkanı" → sıra sayıları kesme ile
- "saat 16.00 sıralarında" → saat ifadesi
- Tarih: "20 Mayıs 2026" veya "bu sabah", "dün akşam"

## HABER METNİ YAPISI

### Açılış Cümlesi (Lead) — ALTIN KURAL
İlk cümle olayın tüm özünü içerir:
```
DOĞRU: "Kaza saat 16.00 sıralarında Mimarsinan OSB 1'nci Cadde'de meydana geldi."
DOĞRU: "Kayseri merkezli 19 ilde düzenlenen yasa dışı bahis operasyonlarında 94'ü ilimizde olmak üzere toplam 119 şüpheli yakalandı."
YANLIŞ: "Kayseri'de önemli bir olay yaşandı. Bu olay tüm şehirde yankı uyandırdı."
```

### Paragraf Yapısı
**1. paragraf:** Kim, ne, nerede, ne zaman (lead)
**2. paragraf:** Nasıl oldu / detaylar
**3. paragraf:** Resmi açıklama / kurum bilgisi
**4. paragraf (varsa):** Soruşturma / sonuç durumu

### Geçiş İfadeleri (Standart Türk Haber Dili)
- "Öte yandan..." → ek bilgi eklerken
- "Edinilen bilgiye göre..." → resmi olmayan bilgi
- "Olay yerinde yapılan incelemede..." → trafik/asayiş
- "Soruşturma sürdürülmektedir." → kapanış
- "Gözaltına alınan şahıslar sorgulanmak üzere..." → standart ifade

## KATEGORİ BAZLI YAZIM REHBERİ

### 🚗 Trafik / Kaza Haberleri
**Şablon:**
```
Kaza saat [SAAT] sıralarında [YER]'de meydana geldi. [ARAÇ1], [durum]. Kazanın etkisiyle [ARAÇ2] [sonuç]. Yaralılar olay yerine sevk edilen 112 ekipleri tarafından [hastane adı] Hastanesine kaldırılarak tedavi altına alındı. Kaza ile ilgili soruşturma başlatıldı.
```
**Zorunlu unsurlar:** Saat, yer (cadde/sokak adı), araç türleri, yaralı sayısı ve durumu (hafif/ağır), hastane
**Yasak:** Kaza nedenine dair spekülasyon ("sürücü dikkatsizliği nedeniyle" — soruşturma tamamlanmadan)

### 🚔 Asayiş / Operasyon Haberleri
**Şablon:**
```
[Kurum adı] tarafından [operasyon adı] kapsamında yürütülen çalışmalarda [sayı] şüpheli şahıs gözaltına alındı. [Operasyon detayı]. Gözaltına alınan şahıslar sorgulanmak üzere [merkez]e sevk edildi. Soruşturma sürdürülmektedir.
```
**Zorunlu unsurlar:** Hangi birim (emniyet/jandarma/narkotik), operasyon adı, gözaltı/tutuklama sayısı
**Yasak:** Sanığı suçlu ilan etmek ("suçlu", "katil") — "şüpheli", "sanık" kullanılır

### 👤 Konuşma / Açıklama Haberleri
**Şablon:**
```
[Unvan + Ad Soyad], [program/toplantı adı]nda açıklamalarda bulundu. [Kişi], "[Alıntı]" dedi.
[İkinci alıntı veya ek bilgi varsa]: [Kişi], ayrıca "[Alıntı]" ifadelerini kullandı.
```
**Zorunlu unsurlar:** Tam unvan (28'nci TBMM Başkanı), alıntıların tırnak içinde verilmesi
**Format:** Alıntı tırnak içinde, "dedi." ile biter; uzun alıntılarda "ifadelerini kullandı."

### 🏛️ Belediye / Kurum Haberleri
**Şablon:**
```
[Belediye/Kurum adı], [faaliyet/proje]yle ilgili açıklama yaptı. [Detay]. [Yetkili adı], konuya ilişkin "[alıntı]" dedi.
```

### ⚖️ Adliye / Yargı Haberleri
**Şablon:**
```
[Suç adı] suçundan yargılanan sanık [Ad/başlangıç harfi.], [mahkeme adı]ndaki duruşmada [ne söyledi/ne oldu]. [Sonuç/karar].
```
**Not:** Sanık adı verilmiyorsa "söz konusu şahıs" kullanılır

## SEO BAŞLIK KALIPLARI (kayserim.net + Ahrefs verisi)

**Yüksek trafik getiren formatlar:**
```
"Kayseri'de [konu] son dakika: [detay]"
"Kayseri [konu]nda [gelişme]"
"[Sayı] [kişi/araç] [fiil]: Kayseri'de [konu]"
"Kayseri merkezli [operasyon]: [sayı] gözaltı"
```

**Kategori bazlı SEO kalıpları:**
- Trafik: "Kayseri'de trafik kazası: [detay]"
- Asayiş: "Kayseri'de [suç türü]: [sayı] şüpheli"
- Ekonomi: "Kayseri altın fiyatları [tarih]: [rakam]"
- Belediye: "Kayseri Büyükşehir [faaliyet]"

## METİN UZUNLUĞU
- **Minimum:** 150 kelime (kayserim.net standartı)
- **Optimal:** 250-400 kelime
- **Kısa haber (breaking):** 80-120 kelime kabul edilir
- Uzunluk sayı ile değil, içerikle sağlanır — dolgu yasak

## ÖRNEK DÖNÜŞÜM

### 1ha'dan Gelen Ham Metin:
> "Kayseri Mimarsinan OSB'de meydana gelen trafik kazasında bir otomobil kamyonun altına girmiştir. Kazada 3 kişi hafif yaralanmıştır. İtfaiye ekipleri tarafından yaralılar kurtarılmış ve ambulanslarla hastaneye götürülmüştür."

### Claude'un Üretmesi Gereken Metin:
> Kaza saat 16.00 sıralarında Mimarsinan Organize Sanayi Bölgesi 1'nci Cadde üzerindeki kavşakta meydana geldi. Seyir halindeki bir otomobil, aynı yönde ilerleyen kamyona arkadan çarparak kamyonun altına girdi.
>
> Kazada araçta bulunan 3 kişi hafif yaralandı. Olay yerine sevk edilen itfaiye ve 112 acil servis ekipleri, araçta sıkışan yaralıları kurtararak Kayseri Eğitim ve Araştırma Hastanesi'ne kaldırdı. Yaralıların hayati tehlikesinin bulunmadığı öğrenildi.
>
> Kaza nedeniyle bölgede kısa süreli trafik akışı yavaşladı. Trafik ekipleri gerekli düzenlemeleri yaparak normal seyrine kavuşmasını sağladı. Kazayla ilgili soruşturma başlatıldı.
